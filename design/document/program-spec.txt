⏺ 프로그램 설계서                                                                                                                                                                                                                                                                                                                                                 
                                                                                                                                  
  ---
  1. 개요                                                                                                                                                                                                                                                                                                                                                         
   
  네이버/다음 뉴스에서 키워드로 기사를 자동 수집하고, GPT AI가 내용을 분석하여 분류한 뒤 엑셀로 출력하는 프로그램.                                                                                                                                                                                                                                                
                  
  ---
  2. 기술 스택

  ┌──────────────────┬─────────────────────────────────────┐
  │       역할       │              사용 기술              │
  ├──────────────────┼─────────────────────────────────────┤
  │ UI               │ Streamlit (브라우저에서 실행, 로컬) │
  ├──────────────────┼─────────────────────────────────────┤
  │ 언어             │ Python                              │
  ├──────────────────┼─────────────────────────────────────┤
  │ 입력 파일 파싱   │ python-docx                         │
  ├──────────────────┼─────────────────────────────────────┤
  │ 네이버 뉴스 수집 │ 네이버 검색 API                     │
  ├──────────────────┼─────────────────────────────────────┤
  │ 다음 뉴스 수집   │ requests + BeautifulSoup (크롤링)   │
  ├──────────────────┼─────────────────────────────────────┤
  │ 기사 분류        │ OpenAI API (GPT-4o-mini)            │
  ├──────────────────┼─────────────────────────────────────┤
  │ 엑셀 출력        │ openpyxl                            │
  ├──────────────────┼─────────────────────────────────────┤
  │ API 키 관리      │ .env 파일 (python-dotenv)           │
  ├──────────────────┼─────────────────────────────────────┤
  │ 코드 관리        │ GitHub                              │
  └──────────────────┴─────────────────────────────────────┘

  3. 프로젝트 폴더 구조

  search-assistant/
  ├── .env                  ← API 키 저장 (GitHub에 올리지 않음)
  ├── .gitignore            ← .env 제외 설정
  ├── requirements.txt      ← 설치 목록
  ├── app.py                ← Streamlit 메인 화면
  └── modules/
      ├── file_parser.py    ← .docx 파일 파싱
      ├── naver_search.py   ← 네이버 기사 수집
      ├── daum_search.py    ← 다음 기사 수집
      ├── classifier.py     ← GPT 분류
      └── excel_writer.py   ← 엑셀 파일 생성

  ---
  4. 입력 파일 (.docx) 형식

  1. 키워드
  삼성전자, 이재용, 가격

  2. 분류 기준
  - 일람: 키워드가 포함된 기사 전부
  - 단순언급: 단순히 키워드가 포함되어 있고 다른 기준에 부합하지 않는 경우
  - 부정적: 키워드에 대해 네거티브한 언급이 있는 경우
  - 단독: 단독 기사인 경우

  ---
  5. 실행 흐름

  [1] .docx 파일 업로드
         ↓ 키워드 + 분류 기준 파싱
  [2] 날짜/시간 범위 선택 (예: 2024-03-25  09:00 ~ 13:00)
         ↓
  [3] 검색 시작 버튼 클릭
         ↓
  [4] 네이버 API + 다음 크롤링으로 기사 수집
         ↓ 날짜/시간 범위 필터링
  [5] 중복 기사 제거 (네이버+다음 중복)
         ↓
  [6] GPT-4o-mini가 기사별 분류
         ↓
  [7] 엑셀 파일 생성 → 다운로드

  ---
  6. 엑셀 출력 형식

  시트 구성: 분류 기준의 항목 수만큼 시트 생성

  ┌──────────┬────────────────────────────────┐
  │  시트명  │              내용              │
  ├──────────┼────────────────────────────────┤
  │ 일람     │ 수집된 모든 기사               │
  ├──────────┼────────────────────────────────┤
  │ 단순언급 │ GPT가 단순언급으로 분류한 기사 │
  ├──────────┼────────────────────────────────┤
  │ 부정적   │ GPT가 부정적으로 분류한 기사   │
  ├──────────┼────────────────────────────────┤
  │ 단독     │ GPT가 단독으로 분류한 기사     │
  └──────────┴────────────────────────────────┘

  각 시트 컬럼:

  ┌────────┬───────────┬────────┬──────────┬──────┬──────────┐
  │ 키워드 │ 날짜/시간 │ 언론사 │ 기사제목 │ 링크 │ 분류이유 │
  ├────────┼───────────┼────────┼──────────┼──────┼──────────┤
  └────────┴───────────┴────────┴──────────┴──────┴──────────┘

  일람 시트에는 모든 기사 포함. 나머지 시트는 중복 없이 1개 카테고리에만 배정.

  ---
  7. API 키 관리 (.env)

  OPENAI_API_KEY=sk-...
  NAVER_CLIENT_ID=...
  NAVER_CLIENT_SECRET=...

  .gitignore에 .env 등록 → GitHub에 절대 올라가지 않음.
